# results/

Full quantitative results for all four model phases.

---

## Segmentation Performance — HC18 Test Set

| Phase | Architecture | Dice (%) | MAE (mm) | Params | Size (MB) |
|-------|-------------|----------|----------|--------|----------|
| Phase 0 | Residual U-Net + deep supervision | **97.75** | **1.65** | 8.11M | 97.5 |
| Phase 4a | Pruned static (hybrid crossover) | 97.64 | 1.76 | 4.57M | 18.3 |
| Phase 2 | Temporal self-attention (16 frames) | 95.95 | 2.10 | 8.90M | 35.7 |
| Phase 4b | Pruned temporal (backbone only) | 96.00 | 2.06 | 5.20M | 20.9 |

✅ All models pass ISUOG ≤3mm threshold for second-trimester screening biometry.

---

## Ablation Study (Temporal Attention)

| Config | Architecture | Dice (%) | MAE (mm) | Δ vs Config A |
|--------|-------------|----------|----------|---------------|
| A | Phase 0 static single frame | 97.36 | 1.75 | — |
| B | 16-frame cine, no attention | 81.48 | 19.37 | −15.88pp / +17.62mm |
| **C** | **16-frame cine + temporal attention** | **95.71** | **2.10** | **−1.65pp / +0.35mm** |

Config B → C recovery: +14.23pp Dice, −17.27mm MAE (93% of gap to Config A recovered).

---

## Compression Results

| Pruning Step | Original Params | Pruned Params | Compression | Δ Dice | Δ MAE |
|-------------|----------------|---------------|-------------|--------|-------|
| Phase 0 → 4a | 8.11M | 4.57M | **−43.7%** | −0.11pp | +0.11mm |
| Phase 2 → 4b | 8.90M | 5.20M | **−41.6%** | +0.05pp | −0.04mm |

Phase 4b Dice and MAE both improve after pruning — compression acts as regularization on the backbone without affecting the temporal attention module.
