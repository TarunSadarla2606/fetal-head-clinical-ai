# results/

Full quantitative results for all phases. All models evaluated on HC18 test set (Radboud UMC).

---

## Segmentation Performance

| Phase | Architecture | Dice (%) | MAE (mm) | Params | Checkpoint size |
|-------|-------------|----------|----------|--------|----------------|
| Phase 0 | Residual U-Net + deep supervision | **97.75** | **1.65** | 8.11M | 97.5 MB |
| Phase 4a | Phase 0 — Hybrid Crossover pruned | 97.64 | 1.76 | 4.57M | 18.3 MB |
| Phase 2 | Temporal self-attention (16 frames) | 95.95 | 2.10 | 8.90M | 35.7 MB |
| Phase 4b | Phase 2 — backbone pruned, TAM intact | 96.00 | 2.06 | 5.20M | 20.9 MB |

✅ All models pass ISUOG ≤3 mm threshold for second-trimester biometry.

Phase 2 training detail: Val Dice 96.29% (best checkpoint); Test Dice 95.95%.  
Phase 0 used as Phase 2 backbone initialisation (reference: Dice 97.36%, MAE 1.75 mm).

---

## Ablation Study (Temporal Attention)

| Config | Architecture | Dice (%) | MAE (mm) | Δ Dice vs A | Δ MAE vs A |
|--------|-------------|----------|----------|------------|------------|
| A | Phase 0 static single frame | 97.36 | 1.75 | — | — |
| B | 16-frame cine, identity attention (no TAM) | 81.48 | 19.37 | −15.88pp | +17.62mm |
| **C** | **16-frame cine + temporal attention (Phase 2)** | **95.95** | **2.10** | **−1.41pp** | **+0.35mm** |

B→C recovery: +14.47pp Dice, −17.27mm MAE (93% of gap to Config A recovered by TAM alone).

## Compression Results

| Step | Original | Pruned | Compression | Δ Dice | Δ MAE | Wilcoxon p |
|------|----------|--------|-------------|--------|-------|------------|
| Phase 0 → 4a | 8.11M | 4.57M | **−43.7%** | −0.11pp | +0.11mm | 0.0049 (sig., negligible magnitude) |
| Phase 2 → 4b | 8.90M | 5.20M | **−41.6%** | +0.05pp | −0.04mm | 0.1013 (NS) |

Phase 4b accuracy **improves** after pruning (FT recovery rate 103.8%) — compression acts  
as implicit regularization on the backbone without degrading the temporal attention module.

**Final pruned channel counts (Phase 4a):**  
`enc3: 128→71 · enc4: 256→129 · bottleneck: 512→257 · dec4: 256→129 · dec3: 128→65`

## Phase 3 Bias Audit Summary

| Subgroup | Dice trend | MAE trend | Notes |
|----------|-----------|-----------|-------|
| Early GA (<20w) | ↓ slightly | ↑ slightly | Smaller skull, lower SNR |
| Mid GA (20–30w) | **Best** | **Best** | Optimal 2nd-trimester window |
| Late GA (>30w) | ↓ slightly | ↑ slightly | Increasing acoustic shadowing |
| Low image quality | ↓ | ↑ | Consistent with clinical expectation |
| HC size range | No bias | No bias | No systematic over/under-measurement |

Limitation: HC18 has no demographic metadata — true demographic subgroup analysis requires multi-centre data.
